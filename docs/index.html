Scala で始める圏論入門

本サイトは、圏論初心者が圏論について学びながら作成した、Scala プログラマのための入門書です。教科書は Bartosz Milewski 氏著の Category Theory for Programmers の Scala Edition で、構成も原則これに沿っています。

Scala をやっていて、圏論について知りたい・学ぶ土台を作りたいという方の参考になれば幸いです。
